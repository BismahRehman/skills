# Daily Productivity Claude Skills

These are 3 reusable Claude skills to automate repetitive daily tasks, save time, and reduce mental load.

---

## 1. Daily Text Fixer
**Purpose:** Fix grammar and improve professional/polite tone for emails, messages, or posts.  
**Input:** Any text  
**Output:** Clean, professional English, ready to send  
**Time saved:** 5–10 minutes → 20–30 seconds

---

## 2. Quick Reply Generator
**Purpose:** Generate polite, confident, short replies for emails, messages, or LinkedIn.  
**Input:** Received message  
**Output:** 1–3 line ready-to-send reply  
**Time saved:** 5 minutes → 20 seconds

---

## 3. Text Shortener
**Purpose:** Turn long text (CV points, job forms, LinkedIn posts) into concise bullets.  
**Input:** Long paragraph  
**Output:** 2–3 professional bullet points  
**Time saved:** 10 minutes → 30 seconds

---

## How to use
1. Copy the input prompt for the skill.  
2. Replace `{{TEXT}}` or `{{MESSAGE}}` with your content.  
3. Run the skill in Claude or Claude CLI.  
4. Copy the output and use it directly.

---

## Outcome
- Reduces repetitive work  
- Improves text quality  
- Saves significant time daily
